= Deleting portsets
:icons: font
:imagesdir: ../media/

[.lead]
You can use System Manager to delete a portset when it is no longer required.

.Steps

. Click *Storage* > *LUNs*.
. In the *Portsets* tab, select one or more portsets and click *Delete*.
. Confirm the deletion by clicking *Delete*.

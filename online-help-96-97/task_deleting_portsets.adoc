= Deleting port sets
:icons: font
:imagesdir: ../media/

[.lead]
You can use System Manager to delete a port set when it is no longer required.

. Click *Storage* > *LUNs*.
. In the *Portsets* tab, select one or more port sets and click *Delete*.
. Confirm the deletion by clicking *Delete*.
